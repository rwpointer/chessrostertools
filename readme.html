What It Does

RoundReady takes a SwissSys or ChessRoster pairings export file and generates three print-ready documents for each section — all in a single run, in the correct order for your workflow.

1

Results Recording Sheet × 2 copies

A compact table listing every board in the section with result circles (1-0, ½-½, 0-1) for the TD to record outcomes. Byes listed separately at the bottom.

2

Alphabetical Player Lookup Sheet × 2 copies

Every player sorted A–Z by last name showing their board number and color assignment. Used for players who can't find their pairing. Automatically paginates for large sections.

3

Pairing Slips

One slip per board, formatted for placement at the board. White player on the left, Black on the right, board number in the center, result circles to mark by hand, and signature lines for both players. Five slips per page, sized to fill a full 8.5×11 sheet so pages can be stacked and cut cleanly.

How to Use

1Export your pairings from SwissSys or ChessRoster as a .txt file.
2Open RoundReady at rwpointer.github.io/chessrostertools.
3Load your file — drag and drop onto the drop zone or click to browse.
4Select sections to include — all sections are checked by default.
5Set options — slips per page, tournament name override, round override, and time control (shown on the alpha sheet header).
6Toggle sheets — use the checkboxes to include or exclude the results sheet and alpha lookup sheet.
7Click Generate — a full print preview appears instantly in the browser.
8Click Print All Sheets and use these settings: Margins → None, Scale → 100%, Headers & Footers → Off.

Input File Format

RoundReady reads the standard SwissSys / ChessRoster tab-delimited pairings export. Multiple sections in a single file are fully supported — each section starts fresh on a new page.

Features

✓Supports multiple sections in one file (K-12, K-6, Extra Games, etc.)
✓Results and alpha sheets printed first (2 copies each), pairing slips follow
✓Blank slip placeholders on partial pages keep cut lines consistent across all pages
✓Pairing slips reordered so stacking the printed pages and cutting gives consecutive boards together
✓Alphabetical lookup sheet paginates automatically for large sections
✓Time control displayed on the alpha lookup sheet header
✓Works entirely in the browser — no installation, no server, no data uploaded anywhere
✓Single HTML file — works offline once the page has loaded

Print Tips

Set print Margins to None and Scale to 100% — do not use "Fit to page."
Turn off Headers and Footers in the print dialog to avoid browser-added text on your sheets.
The dashed lines on pairing slip pages are cut guides. Stack all pages and cut straight through.
Use Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac) to hard-refresh after uploading a new version to GitHub.
